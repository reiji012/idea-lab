# Alk (アルク)

「歩く」をテーマにしたおさんぽ記録アプリ。

## 概要
日々の歩幅や距離などがわかる散歩計機能に加え、ルート検索やマップ活用を中心としたアクティビティログアプリ。

## 機能
- **散歩計機能**: 日々の歩幅、距離、歩数を記録・可視化。
- **ルート検索**: 散歩に適したルートの検索。
- **マップピン機能**: マップ上にピンを立て、写真やメモなどの詳細情報を記録可能。
- **外部連携**: 他のサービス（例: ドラクエウォークなど）との連携機能。

## 技術スタック
- **フレームワーク**: Flutter
- **ターゲット**: iOS / Android (ネイティブ機能活用のため)
