# 台所ノート

長谷川あかりさんのレシピを管理し、冷蔵庫の食材から最適なレシピを検索できるWebアプリケーション

## 概要

「台所ノート」は、ユーザーが気に入った長谷川あかりさんがXや書籍で発信しているレシピを登録し、冷蔵庫にある食材や食べたいメニューを入力することでレシピを検索できるアプリケーションです。

## 主要機能

### 1. レシピ登録
- **Xの投稿から登録**: X投稿のURLを入力し、画像とテキストを保存。画像から材料・作り方を抽出
- **書籍から登録**: 書籍を撮影した画像をアップロードし、材料・作り方を抽出
- **オリジナル登録**: テキストで材料・作り方、画像(最大4枚)を手動登録

### 2. 冷蔵庫管理
- **レシートから登録**: レシート画像をアップロードして在庫を自動登録
- **手動登録**: 食材名を検索して登録。過去の登録履歴から候補を表示

### 3. レシピ検索
- **食材検索**: 複数の食材をマルチセレクトで選択
- **メニュー検索**: カレー、スープ、サラダ、パスタなど大まかなカテゴリで検索

### 4. おすすめレシピ
- **冷蔵庫の食材で作れるレシピ**: 在庫食材を活用できるレシピを提案
- **よく作るメニュー**: 作成回数順にレシピを表示(昇順・降順切替可能)

### 5. 調理履歴
- **レシピ使用記録**: 作った日付と使用したレシピを記録
- **自由記録**: レシピ以外の料理も材料・作り方と共に記録
- **新規レシピ化**: 記録した内容を新しいレシピとして登録可能

## 画面構成

1. **レシピ一覧画面**
   - レシピの新規登録
   - レシピの検索

2. **冷蔵庫の中身画面**
   - 冷蔵庫の中身を登録・管理

3. **記録(食卓の履歴)**
   - レシピを使った記録を残す

## 技術スタック

- **フレームワーク**: Next.js 16 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **リンター**: ESLint
- **パッケージマネージャー**: npm

## セットアップ

### 前提条件
- Node.js 18.x以上
- npm

### インストール

```bash
# 依存パッケージのインストール
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてアプリケーションを確認できます。

### ビルド

```bash
npm run build
```

### 本番環境での起動

```bash
npm start
```
